# 🍕 La Nonna - SaaS para Pizzerías (MVP)

## Descripción

La Nonna es un MVP de un SaaS diseñado para pequeñas y medianas pizzerías.

El objetivo es simplificar la toma de pedidos mediante una interfaz moderna y responsive que permite al cliente generar automáticamente un pedido completo para enviar por WhatsApp.

Actualmente el proyecto representa la primera etapa del producto y está enfocado en validar la experiencia de usuario antes del desarrollo del backend.

---

## Funcionalidades

✅ Catálogo de productos

✅ Categorías dinámicas

✅ Carrito de compras

✅ Modificación de cantidades

✅ Eliminación de productos

✅ Cálculo automático del total

✅ Checkout

✅ Datos del cliente

✅ Delivery o Retiro

✅ Forma de pago

✅ Cálculo de vuelto

✅ Generación automática del pedido para WhatsApp

✅ Diseño Mobile First
